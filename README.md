**Tech Stack:** Deluge • Zoho Cliq API • Workast API • Google Gemini AI

- Channel–space mapping for real-time sync of task updates, status changes, and activity
- Google Gemini AI integration to extract structured task details from natural language, cutting manual task-creation effort by 40%
- Low-latency REST API integration between Cliq and Workast

🔗 [Repository](https://github.com/Mahesh-xr/Workast)
